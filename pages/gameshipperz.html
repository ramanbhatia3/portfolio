Overview

GameShipperz is a comprehensive platform designed for gaming enthusiasts to browse, search, and purchase the latest video games, consoles, and high-performance gaming PCs. Beyond e-commerce, it serves as an information hub with a dedicated articles section for the latest gaming news.

This project was built collaboratively from scratch. As the Team Leader, I coordinated a group of four developers to deliver a fully functional full-stack application within a rigorous 20-day sprint.

Technical Architecture

To ensure a solid foundation and fast performance, we opted for a vanilla frontend coupled with a robust Node.js backend. This approach allowed us to deeply understand DOM manipulation, asynchronous state management, and custom API integration without relying on heavy frontend frameworks.

Key Features

  • Secure Authentication: Implemented a robust user signup and login system using bcrypt for password hashing and JSON Web Tokens (JWT) for secure session management. Protected routes require Bearer token validation.
  • Dynamic Product Catalog: Developed dedicated Mongoose schemas and API endpoints for Games, Consoles, and Gaming PCs. The frontend fetches and renders this data dynamically, utilizing custom CSS loaders for better UX.
  • Advanced Search & Filtering: Built client-side debounced search functionality alongside brand, platform, and price sorting capabilities, allowing users to instantly navigate large product arrays.
  • Interactive Cart Management: Engineered a fully functional cart state tied to the authenticated user's database document. Users can seamlessly add items, view aggregated totals, and remove products with real-time UI updates.
